HDOJ-4267 A Simple Problem with Integers

发表于 2017-03-17 | 分类于 解题报告 ， HDOJ

链接

题意

给定 n 个数， \(A_1,A_2,…,A_n\) ，支持以下两种操作。
1 a b k c 表示对于满足 \(a \leq i \leq b\) 且 \((i-a) \bmod k == 0\) 的 \(A_i\) 加上 c
2 a 查询 \(A_a\) 的值

分析

这个题目看上去就很像是线段树的题目，也确实个线段树。但是不同的是，这个题目不是区间修改，而是修改离散的点。看网上有 dalao 分了55种情况，并没有看懂。
我的做法是，线段树每个节点的 lazy_tag 按 k 分11种情况，然后问题就是如何进行 pushdown 操作？
我比较菜，没有想明白怎么 pushdown ，但是因为是单点查询，所以我们不进行 pushdown ，只将每次 update 的时候修改的区间的 lazy_tag 修改。然后 query 的时候自底向上统计每一层的 lazy_tag 。
需要注意的是 update 中修改右半区间的操作，详见代码。
