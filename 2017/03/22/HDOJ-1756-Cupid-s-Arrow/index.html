HDOJ-1756 Cupid's Arrow

发表于 2017-03-22 | 分类于 解题报告 ， HDOJ

链接

题意

按顺时针方向给出 n 个多边形的顶点。之后给出 m 个点，判断这些是否都在多边形内。

分析

可以用射线法解决这个问题，首先判断给定的点是否在多边形的边上，若不在，引一条射线，判断射线与多边形的交点个数，若为奇数个，则该点在多边形内。
