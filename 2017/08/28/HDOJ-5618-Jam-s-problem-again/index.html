HDOJ-5618 Jam's problem again

发表于 2017-08-28 | 分类于 解题报告 ， HDOJ

链接

题意

给定$n$个坐标$(x_i,y_i,z_i)$，求对于坐标$(x_i,y_i,z_i)$满足$x_j \leq x_i~\&\&~y_j \leq y_i~\&\&~z_j \leq z_i$的$j$的个数。

分析

mlystdcall
首先简单介绍一下CDQ分治，我也是看这这位dalao的博客学习的。
CDQ分治，主要是先进行二分，然后通过$(L, Mid)$更新$(Mid + 1, R)$的答案，即用左边的子问题帮助解决右边的子问题。
这是一个典型的三维偏序问题，可行的解法有很多，我是用了第一维排序，第二维CDQ分治，第三维树状数组维护的方法。
三维偏序问题是基于一维二维偏序问题的，具体详细的内容可以参考mlystdcall这篇博客，这里简单讲一下思路和一些需要注意的地方。
首先我们按照$x$排序，这样我们就可以消除$x$对于答案的影响。
对于第二维我们考虑用CDQ分治解决，考虑归并排序的过程，我们在CDQ的同时进行归并排序，就可以保证$y$的有序性，同时可以用树状数组维护第三维$z$，具体实现就是用两个指针$i, j$分别扫描区间$(L, Mid)$和$(Mid + 1, R)$，如果$a[i].y \leq a[j].y$我们就把$a[i].z$插入树状数组，如果$a[i].y > a[j].y$，说明在左区间中比当前的$a[j].y$小的数已经没有了，即左区间有可能对$a[j]$产生贡献的点已经全部插入树状数组了，我们只需要统计树状数组中小于等于$a[j].z$的数即可。
重复这个过程就能求得全部点的答案了。
P.S.
注意每次归并操作之前都要清空树状数组，但是memset会TLE，所以我是插入操作完成之后再插入一遍-1完成初始化。
CDQ分治只处理了排序后这个点左边的点对于答案的影响，所以如果有重合点的话，会少算一些答案，具体就是在进行CDQ分治之前处理出来相同点右侧对于这个点答案的影响。
注意是$a[i].y \leq a[j].y$进行插入操作，在这一个地方WA的生活不能自理。
