HDOJ-3584 Cube

发表于 2017-04-07 | 分类于 解题报告 ， HDOJ

链接

题意

有一 N*N*N 的三维矩阵 A ，只有 0 和 1 两种取值，开始时每个元素都是 0，定义如下两种操作：
\(1~x_1~y_1~z_1~x_2~y_2~z_2\)表示将满足\(x_1 \leq i \leq x_2 ~ \&\& ~ y_1 \leq j \leq y_2 ~ \&\& ~ z_1 \leq k \leq z_2\)的\( A_{(i,j,k)}\)取反。
\(0~x~y~z\)表示查询现在\(A_{(x,y,z)}\)的值。
